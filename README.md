# Elitem

Fast and secure Messenger Platform

## Getting Started


### Prerequisites

JDK8
Postgresql
Cassandra
```
sudo apt install postgresql
```

### Installing


```
./sbt run
```

## Running the tests

```
./sbt test
```
## Deployment

```
sbt dist
```
